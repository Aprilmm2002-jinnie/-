### 2. 🤖 AIGC 研究：AI 图像生成的偏见分析
*角色：研究员 & 数据分析*

针对 AI 图像生成模型中存在的“多样性与偏见”进行的深度研究，聚焦于“Cute Dog”提示词。

* **研究方法：** 分析了来自 DeepAI, Stable Diffusion, MidJourney 的 **509 个样本**。
* **工具使用：** Gephi (网络分析), Python (数据处理)。
* **主要发现：** 揭示了 AI 系统对浅色犬种的强烈偏好（浅色占比 **77.99%** vs 深色占比 **0.39%**）。
* **视觉产出：** 制作了“新手创作启航计划”的 AIGC 海报。
